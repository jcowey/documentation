      • cacheKey

        public String cacheKey(String direction,
              String text)
        Generate the cache key for this transformer based on the direction and input text. Uses an MD5 of the input text, and generates keys in the form: grammar_hash:direction:inputmd5 So all of the entries for a grammar hash can be invalidated at once. See: http://jakarta.apache.org/jcs/faq.html#hierarchical-removal
