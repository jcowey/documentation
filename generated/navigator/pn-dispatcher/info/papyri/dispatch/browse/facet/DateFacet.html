Class DateFacet

      info.papyri.dispatch.browse.facet.DateFacet

public class DateFacet
extends Facet

The Facet used for setting and displaying date constraints. Note that unlike the other Facets, DateFacet uses two widgets, and stores two values. This complicates the code internal to each method considerably. Note also that these values (stored in the terminusAfterWhich and terminusBeforeWhich members) are triple-state: they may be '0' (effectively, 'undefined'), 'Unknown', or the String representation of an Integer value.
